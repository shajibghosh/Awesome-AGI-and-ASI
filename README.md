## Contributing Guide

Contributions are welcome! Please follow this format when adding resources:

- **Title** ‚Äì Author(s) (Year). [Link](URL) ‚Ä¢ *Category/Context* ‚Ä¢ One-line description

**Example:**

- **The Alignment Problem** ‚Äì Brian Christian (2020). [Link](https://en.wikipedia.org/wiki/The_Alignment_Problem) ‚Ä¢ *Book* ‚Ä¢ Explores technical and societal challenges of aligning AI systems with human values.

### Guidelines
- ‚úÖ Use **official links** (DOI, arXiv, publisher, GitHub)  
- ‚úÖ Ensure **title, author(s), year** are correct  
- ‚úÖ Keep descriptions **short (1 line)**  
- ‚úÖ Only include resources from **Sept 2020 ‚Üí present**  
- ‚úÖ Avoid duplicates  

### Pull Request steps
1. Fork the repo  
2. Add your resource  
3. Commit (`git commit -m "Add [Resource Title]"`)  
4. Open a Pull Request
